

# 📚 Prompt Directory — DevOps Edition

A curated library of **gold-standard prompts** designed for software development, DevOps, and infrastructure engineering.  
This project provides reusable, structured prompts built with frameworks like **CO-STAR, Few-Shot, and Chain-of-Thought** to accelerate workflows across the software lifecycle.

---

## 🚀 Features

### 🔧 DevOps & Infrastructure
- CI/CD pipeline generator (GitHub Actions / GitLab CI).
- Production-ready Dockerfile creation (multi-stage builds).
- Terraform module skeletons for cloud infrastructure.

### 💻 Development
- Refactor legacy modules to reduce complexity.
- Implement new features following coding standards.

### 📝 Code Review
- Structured PR reviews (correctness, performance, security, readability).
- Architectural reviews for system-level changes.

### 🐞 Debugging & Incident Response
- Root-cause analysis from stack traces.
- Production incident triage using logs, metrics, and deploy history.

### 📖 Documentation
- API endpoint documentation in OpenAPI style.
- Auto-generated README.md files from codebases.

### 🏗️ Architecture
- System design proposals with constraints (scale, latency, consistency).
- Trade-off analysis between two architectures.

### ✅ Testing
- Unit test generation (covering edge cases and error paths).
- End-to-end test plans with flows, assertions, and data strategies.

### 🔒 Security
- STRIDE-based threat modeling for new features.
- Security code audits for vulnerabilities (SQL injection, XSS, weak crypto).

---

## 🎯 Purpose
This repository acts as a **toolkit for engineers**, offering reusable prompts to accelerate:
- CI/CD setup  
- Debugging & incident response  
- Code reviews & documentation  
- Security audits & architecture design
